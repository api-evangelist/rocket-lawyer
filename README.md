# rocket-lawyer

Rocket Lawyer — DIY lawyer-vetted legal documents, electronic signature, and on-call attorney consultations, plus a partner REST API platform for embedding legal-document creation, e-signature, and event-driven workflows into third-party products.

## APIs

| API | Description | OpenAPI |
|---|---|---|
| Authentication API | OAuth 2.0 access / service / scoped tokens for partner apps | [openapi/rocket-lawyer-authentication-api-openapi.yml](openapi/rocket-lawyer-authentication-api-openapi.yml) |
| RocketDocument v2 API | Interview-driven creation of lawyer-vetted legal documents | [openapi/rocket-lawyer-rocketdocument-api-openapi.yml](openapi/rocket-lawyer-rocketdocument-api-openapi.yml) |
| RocketSign & Binders API | Bundle documents into binders and run an e-signature workflow | [openapi/rocket-lawyer-rocketsign-binders-api-openapi.yml](openapi/rocket-lawyer-rocketsign-binders-api-openapi.yml) |
| Events API | Subscribe to and pull tenant activity notifications | [openapi/rocket-lawyer-events-api-openapi.yml](openapi/rocket-lawyer-events-api-openapi.yml) |
